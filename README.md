# univercity-of-macau-ECE-major-resource
# 前言
本仓库旨在为澳大ECE本科专业的同学提供课上的一些ppt和考试相关的资料及个人学习建议，**同时也会提供微电子方向的书籍**

由于本专业自身特殊性和封闭性，在网上很难找到有关的公开资料与资源，因此难以像CS专业成系统性的学习。故为倡导开源精神，在此提供作者本人所能搜集到的资源，希望能够为你的学习提供对应的帮助

## 白色部分
ECEN均为必修课程
## 蓝色部分
RE为大三选修课程
## 绿色部分
eelc为英语课，作者建议各位尽量在高考后考过雅思6.5来免修此课程，不然压力会很大
 （关于免修和add/drop请看code里的[澳大新生入学相关建议](https://github.com/dacintliaoyuan/univercity-of-macau-ECE-resource/tree/main/%E6%BE%B3%E9%97%A8%E5%A4%A7%E5%AD%A6%E6%96%B0%E7%94%9F%E5%85%A5%E5%AD%A6%E7%9B%B8%E5%85%B3%E5%BB%BA%E8%AE%AE)
 
CHLL是中文课，PORT为葡语课，这两个课程是为了给高考语文分数低于一定成绩的人提供的，二选一即可（注意：**请看清楚课程是粤语还是普通话教学**）

cisc是记得去考试就行
## 粉红色部分
体育课和书院课，基本上均为大一大二的课程
## 棕黄色部分
该部分就是内地的通识课，主要说明以下几点 
1. GEST1004是微积分课，ECE专业大一上必修，这个不能免掉，不然会卡进度
2. GESB1000就是学长学姐口中的“三合一”，这个目前在课表上是GESB1017
3. 最下面的GE是选择GESB，GELH，GEGA这三个领域中的任意一个课程即可（三个都要选择一个）

## 课程吐槽/评价

### ECEN1011
我说韋sir神中神，有没有懂的

课上认真听讲，不懂的知识点下课去问，这个课尽量不要旷课，不然你会跟不上的

考试的内容跟他复习课的ppt上面的内容差不多，自己注意留意一下

### ECEN1012
这门课教授是FEN HOU,人称ppt战神）

这个课上学期讲计网，下学期讲c语言。个人建议是上课看她ppt自学，然后下课对不懂的知识点询问

考试内容基本上就是往年卷子+assignment+lab，期中和期末的卷子我已经上传了，需要的同学请自行查询
### ECEN1001
要做pre，我这个课基本上都在底下打音游，没啥建议）

补充：pre占比很高，要认真找好组员并且完成
### GEST1004
微积分，前半部分是导数，后面是微积分内容，我个人建议提前用**英语学习**，但是不要用内地教材（中文的术语太繁杂了）

推荐资料中的课 + MIT的微积分课程的题（cs自学指南里面有收录）

### 英语课
关于英语课，我个人建议是**尽早考雅思或者drop**，本身这个课程给分就不好还有早八，纯粹的折磨
